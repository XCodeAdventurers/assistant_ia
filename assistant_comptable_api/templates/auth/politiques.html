Politique de confidentialité

Informations que nous collectons

Lorsque vous utilisez notre application, nous pouvons collecter les types d'informations suivants :

  • Informations personnelles : telles que votre nom, adresse e-mail, numéro de téléphone, sexe, etc.
  • Informations sur les transactions : détails des transactions financières effectuées dans l'application.
  • Informations de compte : détails de votre compte bancaire, soldes, types de comptes, etc.
  • Informations d'utilisation : interactions avec l'application, pages visitées, fonctionnalités utilisées, etc.

Comment nous utilisons vos informations

Nous utilisons les informations collectées dans le but de fournir, maintenir, protéger et améliorer notre application de gestion financière.

Partage des informations

Nous ne partagerons vos informations personnelles qu'avec des tiers dans les cas suivants :

  • Avec votre consentement explicite.
  • Avec les fournisseurs de services tiers.
  • Pour respecter la loi.

Sécurité des informations

Nous prenons des mesures raisonnables pour protéger vos informations personnelles contre tout accès non autorisé ou utilisation abusive.

Vos choix

Vous avez le droit de choisir comment vos informations sont collectées, utilisées et partagées.
